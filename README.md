### MERN CRUD Rest-API
Goals setter application. Simple project to practice MERN CRUD operation

## Requirements
    create /server/.env then create these env variables.
    ``
    HOST
    PORT
    MONGO_DB_CONNECTION_STRING
    ``

## Run commands
    cd /server
    run -> npm run server
    
    cd /client
    run -> npm run client
